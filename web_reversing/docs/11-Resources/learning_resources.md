## 学习路线建议

### 阶段一：基础夯实（1-2 个月）

| 目标                      | 学习内容                                                                                                                                                                       | 实战项目                                                                                                                |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------- |
| 掌握 Web 基础和开发者工具 | 1. **HTTP 协议**: 《图解 HTTP》<br>2. **JavaScript 基础**: 《JavaScript 高级程序设计》前 10 章<br>3. **开发者工具**: Chrome DevTools 官方文档<br>4. **HTML/CSS**: MDN Web Docs | • 用 Chrome DevTools 分析 10 个不同网站的请求<br>• 完成 JavaScript30 的前 10 个项目<br>• 手写简单的表单验证和 AJAX 请求 |

---

### 阶段二：工具掌握（1-2 个月）

| 目标                         | 学习内容                                                                                                                                                                 | 实战项目                                                                                                      |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------- |
| 熟练使用自动化工具和抓包工具 | 1. **Puppeteer**: 官方文档 + 实战案例<br>2. **抓包工具**: Burp Suite 或 Charles<br>3. **Python 基础**: 廖雪峰 Python 教程<br>4. **正则表达式**: 在线练习（regex101.com） | • 用 Puppeteer 爬取 3 个动态网站<br>• 用 Burp Suite 分析移动 APP 接口<br>• 写一个自动登录脚本（带验证码识别） |

---

### 阶段三：逆向技术（2-3 个月）

| 目标                       | 学习内容                                                                                                                                                                                | 实战项目                                                                                                          |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| 理解加密、混淆、反调试技术 | 1. **JavaScript 进阶**: 《你不知道的 JavaScript》<br>2. **加密算法**: crypto-js 文档 + CyberChef 实践<br>3. **AST 解析**: Babel 插件开发教程<br>4. **反混淆**: webcrack、de4js 工具使用 | • 分析 5 个网站的加密参数（sign、token）<br>• 用 Babel 写一个简单的反混淆脚本<br>• 破解一个简单的 JavaScript 混淆 |

---

### 阶段四：综合实战（持续）

| 目标                   | 学习内容                                                                                                                                                                             | 实战项目                                                                                                                               |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| 解决真实世界的复杂案例 | 1. **高级反爬虫**: TLS 指纹、Canvas 指纹、设备指纹<br>2. **协议逆向**: WebSocket、gRPC、Protobuf<br>3. **WebAssembly**: wabt 工具链、逆向分析<br>4. **移动端**: Frida Hook、APP 脱壳 | • 参加 CTF 比赛（Web 方向）<br>• 逆向一个 Webpack 打包的 SPA 应用<br>• 分析一个使用 WebAssembly 的网站<br>• 完成吾爱破解论坛的逆向题目 |

## 学习技巧

### 高效学习方法

| 方法             | 说明                   |
| ---------------- | ---------------------- |
| **以项目为驱动** | 边学边做，不要只看不练 |
| **主动调试**     | 多用断点，少猜测       |
| **记录笔记**     | 整理常见套路和技巧     |
| **分享交流**     | 写博客、参与讨论       |
| **定期复习**     | 温故知新，巩固知识     |

---

### 常见误区

| 误区                    | 说明                                   |
| ----------------------- | -------------------------------------- |
| ❌ **只看教程不动手**   | 看懂 ≠ 会做                            |
| ❌ **追求工具而非原理** | 工具会过时，原理不会                   |
| ❌ **完美主义**         | 不需要 100% 还原代码，理解核心逻辑即可 |
| ❌ **孤立学习**         | 多与社区交流，避免闭门造车             |
| ❌ **忽视基础**         | HTTP、JavaScript 基础是一切的根基      |
