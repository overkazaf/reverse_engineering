# 05-Case-Studies Summary

本章节通过实战案例来巩固逆向技能。

---

## 💭 开始前的思考

在进入实战案例之前，先思考：

1. **看懂 vs 做到：** 看别人的逆向分析文章，和自己独立完成一个案例，差距有多大？
2. **套路识别：** 市面上成千上万的网站，它们的反爬虫技术真的都不一样吗？能否总结出通用套路？
3. **失败的价值：** 如果花了一天时间还是没找到加密参数，你会怎么办？
4. **道德边界：** 你逆向出了某网站的完整接口，接下来该做什么？不该做什么？

实战案例不仅教会你技术，更重要的是培养逆向思维和职业道德。

---

## 主要内容

- **常见验证码逆向**: 滑块、点选、旋转验证码的参数分析与轨迹模拟。
- **加密参数逆向**: 某电商、某社交平台的签名算法还原 (MD5, AES, RSA, 自定义算法)。
- **数据解密**: Protobuf 数据解析、AES-GCM 解密实战。
