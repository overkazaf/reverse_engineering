# 01-Tooling Summary

本章节介绍了 Web 逆向工程中常用的工具及其配置方法。

---

## 💭 开始前的思考

在深入学习具体工具之前，不妨先思考以下问题：

1. **为什么需要这么多工具？** 浏览器自带的 DevTools 不够用吗？
2. **你遇到过哪些场景** 是浏览器开发者工具无法解决的？
3. **如果你是网站开发者**，会如何防止别人轻易抓包分析你的接口？
4. **工具的本质是什么？** 它们解决了哪些核心问题？

带着这些问题，让我们开始探索工具箱的奥秘。

---

## 主要内容

- **浏览器开发者工具 (DevTools)**: Elements, Console, Sources (断点调试), Network (抓包), Application (存储) 面板的高级用法。
- **抓包工具**:
  - **Charles/Fiddler**: HTTP/HTTPS 抓包、断点修改、重放请求。
  - **Burp Suite**: 强大的渗透测试套件，用于 Repeater (重放), Intruder (爆破), Decoder (编码转换)。
- **代理与 VPN**: 理解正向代理与反向代理，配置系统级代理。
- **环境搭建**: Node.js, Python, Chrome (带参数启动) 等基础环境配置。
