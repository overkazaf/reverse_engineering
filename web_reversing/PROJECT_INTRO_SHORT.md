# Web 逆向工程烹饪食谱 - 项目简介

## 📖 一句话介绍

近日将 Web 逆向工程的学习文档系统化整理成 **Cookbook** 形式，涵盖**基础知识**、**工具使用**、**核心技术**、**高级主题**、**工程实践**及**实战案例**，为逆向学习者提供从入门到进阶的完整知识体系。

---

## ✨ 核心亮点

### 📚 内容全面
- **8 大章节** × **69+ 篇文档** × **10 万+ 字**深度内容
- 从 JavaScript 基础到 WebAssembly，从工具使用到分布式部署
- 涵盖主流逆向工具（Chrome DevTools、Burp Suite、Puppeteer、Babel）原理和实战技巧

### 🎯 实战导向
- **20+ 真实案例**：电商签名、社交加密、视频 DRM、验证码破解
- **200+ 代码示例**：Hook 模板、反混淆脚本、自动化工具
- **50+ 开源项目**推荐，直接上手实践

### 🧠 主动学习
- 每个章节配有**思考题**，引导从被动接受到主动思考
- **渐进式学习路径**：基础 → 工具 → 技术 → 高级 → 工程 → 案例

### 📖 多种阅读方式
- 🌐 **在线文档**（MkDocs）：支持全文搜索、交互式导航
- 📄 **PDF 电子书**：附带目录书签，方便离线阅读

## 🚀 快速开始

```bash
# 克隆仓库
git clone <repository-url> && cd web_reversing

# 安装依赖并启动
pip install -r requirements.txt
mkdocs serve

# 访问 http://127.0.0.1:8000
```

## ⚠️ 使用说明

### PDF 转换
PDF 转换脚本在处理**中文格式**和**表格排版**时存在一些已知 bug（如排版错位、表格溢出）。建议：
1. 下载 PDF 后开启**目录模式**（书签/大纲）
2. 利用目录快速跳转到感兴趣的章节
3. 或直接使用在线文档（体验更佳）

我们正在持续优化转换效果，欢迎反馈问题！
