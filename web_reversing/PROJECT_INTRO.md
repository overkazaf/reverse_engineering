# Web 逆向工程烹饪食谱 (Web Reversing Cookbook)

## 📖 项目简介

近日将 Web 逆向工程的学习文档系统化整理成 **Cookbook** 形式，旨在为逆向工程学习者提供一份全面、结构化、易于理解的实战手册。本项目内容涵盖**基础知识**、**工具使用**、**核心技术**、**高级主题**、**工程实践**以及**真实案例**，力求打造一个从入门到进阶的完整学习路径。

---

## ✨ 核心特色

### 1️⃣ 系统化知识体系
- **8 大章节**，**69+ 篇文档**，超过 **10 万字**的深度内容
- 从 HTTP 协议到 WebAssembly，从工具使用到工程部署，全方位覆盖
- 每个章节都配有**思考题**，引导读者主动思维，而非被动接受

### 2️⃣ 实战导向
- 涵盖**主流逆向工具**原理和使用技巧（Chrome DevTools、Burp Suite、Puppeteer、Babel 等）
- 提供**真实案例分析**（电商签名、社交媒体加密、视频网站 DRM、验证码破解等）
- 包含**可执行脚本**（Hook 模板、反混淆工具、自动化脚本等）

### 3️⃣ 渐进式学习路径
- **00-Foundations**: JavaScript、HTTP/HTTPS、浏览器架构等基础知识
- **01-Tooling**: DevTools、抓包工具、AST 解析工具使用指南
- **02-Techniques**: Hook 技术、反混淆、调试技巧、加密定位
- **03-Advanced-Topics**: 反调试、指纹识别、WebAssembly、JSVMP、前端加固
- **04-Engineering**: RPC 调用、自动化框架、分布式部署、监控告警
- **05-Case-Studies**: 电商、社交、金融、视频等领域实战案例
- **06-Scripts**: JavaScript Hook、反混淆脚本、自动化工具集
- **07-Others**: 学习资源、开源项目、FAQ、法律边界

### 4️⃣ 多种阅读方式
- 📱 **在线文档**：基于 MkDocs 构建的交互式文档站点，支持全文搜索
- 📄 **PDF 电子书**：支持导出 PDF（附目录书签），方便离线阅读和打印
- 🔖 **Markdown 原文**：所有文档均为 Markdown 格式，方便二次编辑和分享

## 📚 主要内容

### 基础篇（Foundations）
- **JavaScript 核心**：闭包、原型链、异步编程、执行机制
- **网络协议**：HTTP/HTTPS、TLS 握手、CORS、同源策略
- **浏览器原理**：多进程架构、渲染流水线、V8 引擎
- **Web API**：XHR、Fetch、WebSocket、Canvas、Storage

### 工具篇（Tooling）
- **Chrome DevTools**：断点调试、Network 分析、Local Overrides、Snippets
- **抓包工具**：Burp Suite、Charles、mitmproxy、Whistle
- **自动化工具**：Puppeteer、Playwright、Selenium
- **AST 工具**：Babel、@babel/parser、AST Explorer、webcrack

### 技术篇（Techniques）
- **Hook 技术**：XHR Hook、Fetch Hook、Cookie Hook、Prototype Hook、Proxy Hook
- **反混淆**：Obfuscator.io、控制流平坦化、字符串数组、AST 还原
- **调试技巧**：条件断点、XHR 断点、DOM 断点、堆栈跟踪
- **加密定位**：全局搜索、堆栈分析、Hook 拦截、补环境

### 高级篇（Advanced Topics）
- **反调试对抗**：debugger 检测、Function.toString、窗口大小检测、时间差检测
- **指纹识别**：Canvas 指纹、WebGL 指纹、Audio 指纹、TLS 指纹（JA3/JA4）
- **WebAssembly 逆向**：wasm2wat、Wasm 反编译、动态调试、内存 Dump
- **JSVMP 分析**：虚拟机识别、指令集逆向、控制流分析
- **前端加固**：代码混淆、白盒加密、完整性校验、反调试部署

### 工程篇（Engineering）
- **RPC 技术**：Sekiro、JsRPC、远程函数调用
- **自动化框架**：补环境、JSDOM、Playwright Cluster
- **分布式部署**：代理池、指纹池、任务队列、Docker 编排
- **监控告警**：日志采集、性能监控、异常告警、指标可视化

### 案例篇（Case Studies）
- **电商平台**：签名算法逆向、SKU 加密、价格混淆
- **社交媒体**：Token 生成、设备指纹、反爬虫策略
- **视频网站**：DRM 解密、M3U8 解析、Widevine L3
- **验证码**：滑块验证码、点选验证码、行为轨迹模拟

## 🚀 快速开始

### 在线阅读（推荐）

```bash
# 克隆仓库
git clone <repository-url>
cd web_reversing

# 安装依赖
pip install -r requirements.txt

# 启动本地文档服务器
mkdocs serve

# 浏览器访问 http://127.0.0.1:8000
```

### 导出 PDF

```bash
# 使用自带脚本生成 PDF（基于 Playwright）
python docs_to_pdf.py

# PDF 将保存在 output/ 目录
```

**注意**：PDF 转换脚本在处理中文格式和表格时存在一些已知 bug，可能出现排版错位、表格溢出等问题。建议下载 PDF 后开启**目录模式**（书签/大纲），方便快速跳转到感兴趣的章节。我们正在持续优化转换效果。

## ⚠️ 法律声明

**本项目所有内容仅供学习和研究使用，请勿用于非法用途。**

- ✅ **合法场景**：学术研究、安全测试（授权）、CTF 竞赛、自己的网站
- ❌ **非法场景**：未授权访问、窃取数据、恶意攻击、侵犯知识产权

**使用者应当**：
1. 遵守当地法律法规和网络安全法
2. 尊重网站的 robots.txt 和服务条款
3. 设置合理的请求频率，避免对目标服务器造成压力
4. 不要爬取和传播敏感个人信息

**技术无罪，但使用技术的方式决定了你的责任。请负责任地使用本项目的知识。**
