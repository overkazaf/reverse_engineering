| ---------------- | ---------------------- |
| **以项目为驱动** | 边学边做，不要只看不练 |
| **主动调试** | 多用断点，少猜测 |
| **记录笔记** | 整理常见套路和技巧 |
| **分享交流** | 写博客、参与讨论 |
| **定期复习** | 温故知新，巩固知识 |

---

### 常见误区

| 误区 | 说明 |
| ----------------------- | -------------------------------------- |
| ❌ **只看教程不动手** | 看懂 ≠ 会做 |
| ❌ **追求工具而非原理** | 工具会过时，原理不会 |
| ❌ **完美主义** | 不需要 100% 还原代码，理解核心逻辑即可 |
| ❌ **孤立学习** | 多与社区交流，避免闭门造车 |
| ❌ **忽视基础** | HTTP、JavaScript 基础是一切的根基 |
